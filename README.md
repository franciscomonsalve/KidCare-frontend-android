# KidCare Frontend Android

Aplicación móvil Android desarrollada en Kotlin + Jetpack Compose para la plataforma KidCare.

## Descripción

KidCare es una aplicación orientada a la gestión y seguimiento infantil, permitiendo autenticación de usuarios, registro y acceso diferenciado según roles.

Actualmente el proyecto incluye:

- Login conectado a backend
- Registro de usuarios
- Manejo de sesión con JWT
- Navegación entre pantallas
- Integración con API REST usando Retrofit

---

## Tecnologías utilizadas

- Kotlin
- Jetpack Compose
- Navigation Compose
- Retrofit 2
- Gson Converter
- OkHttp
- Android Studio
- Docker (backend)
- MySQL

---

## Configuración del proyecto

### Requisitos

- Android Studio Hedgehog o superior
- JDK 17
- Backend KidCare ejecutándose en puerto `8081`
